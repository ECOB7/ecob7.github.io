# Portfólio - Enzo Cristopher Oliveira Bittencourt

Portfólio estático para publicar rápido no GitHub Pages.

## Como editar

1. Abra `script.js`.
2. Troque os dados do objeto `portfolio`.
3. Coloque seus links reais de GitHub, LinkedIn, email e projetos.
4. Se quiser trocar a imagem principal, substitua `assets/portfolio-hero.png`.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie estes arquivos para a branch `main`.
3. No GitHub, abra `Settings` > `Pages`.
4. Em `Build and deployment`, escolha `Deploy from a branch`.
5. Selecione `main` e `/root`.
6. Salve e aguarde o link ficar disponível.

## Informações para preencher

Primeira versão preenchida com dados profissionais, habilidades, idiomas, projetos e cursos. Para melhorar a apresentação, ainda vale adicionar:

- Links dos repositórios de cada projeto.
- Links de deploy, quando existirem.
- Prints ou vídeos curtos dos projetos.
- Foto profissional ou imagem alternativa para o topo, se preferir.
